
# LLM Hallucination Detector

A lightweight AI evaluation tool that analyzes large language model outputs for hallucination risk and factual reliability.

This project demonstrates prompt-based AI evaluation workflows used in modern AI governance and safety pipelines.

## Features

• LLM question answering  
• hallucination risk evaluation  
• confidence scoring  
• human-in-the-loop validation

## Technologies

Python  
LLM APIs  
Prompt engineering  
AI evaluation pipelines
